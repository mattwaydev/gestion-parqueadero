# El Paso 🅿️
**GG, encontraste parqueadero**

Sistema de gestión de parqueadero desarrollado en Python con interfaz gráfica.

## Integrantes
- Mateo Carrascal
- Samuel Ramirez

## Tecnologías
- Python 3.14
- CustomTkinter

## Funcionalidades
- Registro y retiro de vehículos
- Asignación aleatoria de espacios
- Mapa visual del parqueadero
- Panel de administrador
- Historial de pagos
- Estadísticas del parqueadero
- Búsqueda por tipo de vehículo

## Cómo ejecutar
1. Instalar dependencias: `pip install customtkinter`
2. Ejecutar: `python intefaz.py`
